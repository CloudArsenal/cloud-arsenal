Cloud Consoles

See everything. Fix faster.

Browser-native consoles for AWS and Azure. Real-time scanning, multi-account support, plugin scanners — one command to deploy.

🟠

AWS CloudOps Console

Scan your entire AWS Organization in real time. Multi-account support, SSO authentication, and a plugin scanner system you can extend with your own scripts.

  • Multi-account scanning via AWS SSO or IAM
  • Live log streaming — see every account as it scans
  • Plugin scanners: KMS, EC2, ALB, Cost Explorer
  • Excel / CSV export + optional S3 scan history
  • One command deploy: docker-compose up
↗ Live on GitHub
🔵

Azure CloudOps Console

Scan all your Azure subscriptions from a single browser session. Entra ID device flow or Service Principal auth — no CLI, no PowerShell required.

  • Multi-subscription scanning via Entra ID or SP
  • Auto-discover subscriptions from your tenant
  • Plugin scanners: orphaned disks, NICs, idle VMs
  • Excel / CSV export + optional Blob Storage history
  • One command deploy: docker-compose up

Infrastructure Toolkits

Deploy. Automate. Repeat.

AWS WorkSpaces Applications

Stream Windows apps or a full desktop to any user via browser — no VPN, no client install. Terraform + Python toolkit for remote training, demos, and contractor access.

  • APP mode (individual windows) or DESKTOP mode
  • On-Demand fleet — zero cost when stopped
  • SSO, IAM keys, or Assume Role auth
  • Private subnet only — no public exposure
  • One tfvars file per customer to replicate
